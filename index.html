Personal Projects

- Snake 2P: As my first project with the game engine Processing, Snake 2P takes directional inputs from two players to create movable snakes. Works using class instantiation and continuously refreshed arrays with input processing

Through countless iterations, by being given training examples and using back-propagation, it is able to learn this difference
